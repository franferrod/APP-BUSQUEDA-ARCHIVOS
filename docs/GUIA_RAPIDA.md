# 🚀 Guía Rápida - Buscador de Piezas ALSI

Bienvenido a tu nueva herramienta para encontrar planos y diseños al instante. Aquí tienes lo básico para empezar:

## 1. ¿Cómo busco algo?
Simplemente escribe en la barra superior lo que necesitas.
*   **Ejemplo**: `travesaño`
*   **Truco Pro**: Puedes buscar varias cosas a la vez separándolas por comas.
    *   Ejemplo: `mesa, inox, 2024` (Buscará archivos que tengan esas tres palabras).

## 2. Filtros Inteligentes (Barra Izquierda)
Si te salen demasiados resultados, usa la barra izquierda para afinar:
```markdown
*   **Compañeros**: ¿Quién diseñó la pieza? (Pablo, Jesús, Marcos, Paco, Dani, Emilia, etc.)
```
*   **Años**: ¿De qué año es el proyecto?
```markdown
*   **Clientes**: Filtra por el cliente final.
*   **Búsqueda en Filtros**: No pierdas tiempo haciendo scroll; escribe el nombre del cliente o proyecto en el cuadro del filtro para localizarlo y seleccionarlo al instante.
```

> **Nota**: Los filtros funcionan en cascada. Si eliges el año "2024", solo verás los clientes y proyectos de ese año.

## 3. ¿Qué hago con los resultados?
Una vez encuentres tu archivo en la lista:
```markdown
*   **Un clic**: Verás una **vista previa** instantánea a la derecha (Nombre, Ruta y Tamaño).
```
```markdown
*   **Doble clic**: Abre la carpeta de Windows donde se encuentra el archivo y lo selecciona.
```
*   **Botón Derecho**: Te permite "Abrir la carpeta" donde está guardado o copiar su ruta.

## 4. Mantenerlo al día
Si acabas de guardar un archivo y no aparece:
1.  Pulsa el botón naranja **"Indexar Compañeros"** (abajo a la izquierda).
2.  Dale a **"Iniciar"**.
3.  ¡Listo! En unos segundos tu archivo aparecerá.
