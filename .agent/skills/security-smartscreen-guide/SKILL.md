# Skill: security-smartscreen-guide
Descripción: Guía para manejar advertencias de Windows SmartScreen en aplicaciones internas.

## Instrucciones para el Usuario
Cuando aparezca el cuadro azul de "Windows protegió su PC":
1. Haz clic en el enlace **"Más información"** que aparece bajo el texto principal.
2. Aparecerá un nuevo botón llamado **"Ejecutar de todas formas"**. Haz clic en él.
3. El programa se abrirá normalmente y Windows recordará tu elección para la próxima vez.

> [!TIP]
> Esta advertencia ocurre porque la aplicación no tiene una firma digital comercial. Como es una herramienta de desarrollo interno de ALSI, es seguro proceder.
